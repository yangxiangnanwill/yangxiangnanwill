字符串实例方法

Java 8 向 String 类中添加了许多实例方法，包括：

  • isEmpty()：判断字符串是否为空。
  • chars()：返回一个 IntStream，其中包含此字符串中的字符。
  • codePoints()：返回一个 IntStream，其中包含此字符串中的 Unicode 代码点。
